# Épico 3: Interface Operacional (Frontend React / Vite)

**Objetivo:** Criar a "cara" do projeto. Uma interface polida, estética e com excelente usabilidade, permitindo o upload fácil e a visualização clara dos resultados.

## História 3.1: Tela de Upload Premium (Drag & Drop)
**Descrição:** Como usuário, quero uma tela intuitiva onde eu possa simplesmente arrastar o meu PDF da prestação de contas do mês para iniciar o processo.

## História 3.2: Feedback Visual Assíncrono
**Descrição:** Como usuário, quero ver o progresso do backend ("Baixando anexos...", "Minerando valores...") para não achar que o sistema travou.

## História 3.3: Dashboard de Resultados e Auditoria
**Descrição:** Como usuário, quero visualizar uma tabela clara com todos os lançamentos extraídos, com destaque visual em vermelho para qualquer linha que contenha uma inconsistência financeira.

## História 3.4: Exportação para Power BI (CSV)
**Descrição:** Como usuário, quero apertar um botão para baixar os dados tratados em um formato pronto para ser "engolido" pelo Power BI.
